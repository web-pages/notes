Atom Editor

Installing/Upgrading

For linux, installing and upgrading are identical:

  1. Go to the atom releasesL1 site and download the latest atom-amd64.deb file
  2. Install/Upgrade by doing the following: sudo dpkg -i atom-amd64.deb

Installing Plugins

To install plugins (called packages in atom):

  1. Go to the Edit drop down menu
  2. Select Preferences
  3. On the left side menu, select Install (it is the last option)

Search and install the desired package.
