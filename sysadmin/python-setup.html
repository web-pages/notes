Working With Python

In order to work with python, use a virtual environment. This requires installing virtualenv and virtualenvwrapper. The latter gives extra commands to virutalenv

Setup Virtual Environments

Install Distribute and Pip

1
2
3
curl -O http://python-distribute.org/distribute_setup.py
sudo python distribute_setup.py
sudo easy_install pip

Install VirtualEnv And VirutalEnvwrapper

1
2
sudo pip install virtualenv
sudo pip install virtualenvwrapper

Setup VirtualEnv

1
2
3
export WORKON_HOME=~/.virtualenv
mkdir $WORKON_HOME
echo "export WORKON_HOME=$WORKON_HOME" >> ~/.bashrc

Setup VirtualEnvWrapper

1
echo "source /usr/local/bin/virtualenvwrapper.sh" >> ~/.bashrc

Extra tips and tricks:

Ensures that if pip creates an extra virtual environment, it is also placed in our WORKON_HOME directory:

1
echo "export PIP_VIRTUALENV_BASE=$WORKON_HOME" >> ~/.bashrc

Creating Virtual Environments

  1. Obtain the version of the python executable that is needed (if the defaults are not wanted):
  2. Use the deadsnakesL1 PPA to get the required version
  3. Manually get what version is wanted:
    • See https://www.python.org/downloads/L2 for the version that is wanted
    • Use wget to get the wanted version, for example, wget http://python.org/ftp/python/2.7.5/Python-2.7.5.tgz
    • Untar, configure and make
  4. Create a new virtual environment with a new version of Python:
  5. The default python executable is always chosen by default. To change this, use the -p
1
mkvirutalenv -p /usr/bin/python2.6 env_name

Quick Command Reference

  • lsvirtualenv - lists all installed virtual environments.
  • rmvirtualenv - will remove a virtual environment.
