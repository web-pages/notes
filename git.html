Config

  • git config push.default current - push the current branch to update a branch with the same name on the receiving end. Works in both central and non-central workflows. This will create a remote branch if it does not already exist
  • git config remote.pushdefault current - will push the branch to the origin remote by default (if not specified) and create if its not present

Git Commands

Fixup and Autosquash

Squash last commit with a combination of fixup and autosquash1

Fixup

1
2
3
4
5
6
$ git add featureA
$ git commit -m "Feature A is done"
[dev fb2f677] Feature A is done
$ git add featureB
$ git commit -m "Feature B is done"
[dev 733e2ff] Feature B is done

If something is added to featureA, use fixup with featureA's commit hash to have commit message copied:

1
2
$ git add featureA
$ git commit --fixup fb2f677

Git has now automatically created a commit message with the word fixup! appended to featureA's original commit message.

Autosquash

Autosquash can now take place (the fixup step above is essential). Pick a hash to rebase, and do the following:

1
$ git rebase -i --autosquash ac5db87

The rebase editor has been opened - just save and quit.
