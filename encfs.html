EncFS

EncFSL11 is an encrypted FUSE file system. It integrates very well with dropbox allowing a local encrypted volume that will be replicated on the cloud. Unlike TrueCrypt, encfs volumes are not a fixed size and can therefore grow and shrink when needed.

Installation

Ubuntu

To install in ubuntu: sudo apt-get install encfs

Mac OS Yosemite

Follow these instructionsL3

Usage

encfs /path/to/encrypted/folder /path/to/decrypted/mount
