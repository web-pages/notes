Find Two Values That Add To A Target In A Binary Tree

  • Published: 26 Feb 2016
Table Of Contents

Problem Statement

Given a binary tree and a target value, find if there are two values that add up to the target.

Solutions

Assume that there are \(n\) nodes in the tree.

No Extra Space

If this is to be accomplished without using any extra space, then the solution would be quadratic. This is because the binary tree must be traversed, and each value in the tree would launch another search for a corresponding value that adds to the target.

  • Time: \(O(n^2)\)
  • Space: \(O(1)\)

Use A Hash Table

Each node of the binary tree would be examined. Assuming the node has value \(x\), we would then search for the value \(target-x\) in the hash table. If this value is not present, then insert the current node into the hash table and continue the traversal.

  • Time: \(O(n)\)
  • Space: \(O(n)\)

unordered_map<int, bool> hash;
bool findValues(int target, Tree* root) {
  if (!root) return false;
  int search = target - root->value;
  if (hash[search] == true) return true;
  hash[root->value] = true;
  return (
    findValues(target, root->left) ||
    findValues(target, root->right)
    );
}
