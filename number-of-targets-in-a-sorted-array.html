Number Of Targets In A Sorted Array

  • Published: 26 Feb 2016

Problem Statement

Given a sorted array with numbers that can be potentially repeated, find the number of occurrences of a target.

Solutions

Linear Search

The easiest way to perform accomplish this task is to keep track of the target while iterating over the array. Assuming the array has \(n\) elements, linear search has the following analysis:

  • Time: \(O(n)\)
  • Space: \(O(1)\)

Binary search

The question did mention that the input array is sorted. Due to this fact, binary search can be used. Using binary search, we would find the first index array that is strictly greater than the target, and then perform another binary search that will find the first index that is greater than or equal to the target. Subtracting the two resulting indexes will give us the answer.

This makes the solution more efficient. Again, assume the array has \(n\) elements:

  • Time: \(O(\log n)\)
  • Space: \(O(1)\)

vector<int> sortedNumbers; // assume this is global
int binarySearch(int target, bool strictlyGreater) {
    unsigned int lo=0; hi=sortedNumbers.size()-1;
    while (lo < hi) {
      int mid = lo + (hi-lo)/2;
      bool result = (strictlyGreater)
        ? (sortedNumbers[mid] > target)
        : (sortedNumbers[mid] >= target);
      if (result)
        hi = mid;
      else
        lo = mid+1;
    }

    return lo;
}

int findOccurrences(int target) {
  return binarySearch(target, true) - binarySearch(target, false);
}
