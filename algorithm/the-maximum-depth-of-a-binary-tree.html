The Maximum Depth Of A Binary Tree

  • Published: 26 Feb 2016
Table Of Contents

Problem Statement

Given a binary tree, find it's maximum depth?

Solution

Recursively traverse the binary tree, keeping track of the depth of the depth of each of it's children. Return the maximum of the two children to the parent (and don't forget to add 1 which represents the depth of the child to the parent).

Analysis

Assume that the binary tree has \(n\) nodes:

  • Time: \(O(n)\)
  • Space: \(O(1)\)

int maxDepth(TreeNode* root) {
  if (root == nullptr) return 0;

  return max(
    maxDepth(root->left),
    maxDepth(root->right)
  ) + 1;
}
